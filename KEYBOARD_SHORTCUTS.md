# ⌨️ Клавіатурні скорочення (Keyboard Shortcuts)

Цей проект використовує адаптивну систему гарячих клавіш, яка залежить від вашої операційної системи.

*   **macOS:** Використовуйте клавішу **Command (⌘)**.
*   **Windows / Linux:** Використовуйте клавішу **Alt**.

---

## 🌍 Глобальні скорочення

Ці комбінації працюють майже з будь-якого місця в додатку (якщо не фокусовано поле вводу).

| Дія | macOS | Windows / Linux |
| :--- | :--- | :--- |
| **Нова транзакція** | `⌘` + `A` | `Alt` + `A` |

---

## 📝 Форми та модальні вікна

Працюють, коли відкрито відповідну форму (Рахунки, Транзакції тощо).

| Дія | macOS | Windows / Linux | Опис |
| :--- | :--- | :--- | :--- |
| **Зберегти / Створити** | `⌘` + `Enter` | `Alt` + `Enter` | Швидке відправлення форми |
| **Зберегти Актив** | `⌘` + `S` | `Alt` + `S` | Специфічно для форми створення активу "на льоту" |
| **Змінити скін (банк)** | `⌘` + `I` | `Alt` + `I` | Відкрити вибір дизайну картки у формі рахунку |
| **Закрити** | `Esc` | `Esc` | Закрити будь-яке модальне вікно |

---

## 💸 Форма транзакції (деталі)

Коли ви перебуваєте у формі створення або редагування транзакції.

| Дія | macOS | Windows / Linux | Опис |
| :--- | :--- | :--- | :--- |
| **Деталі (Товари)** | `⌘` + `D` | `Alt` + `D` | Показати/сховати таблицю товарів (чек) |
| **Додати товар** | `⌘` + `A` | `Alt` + `A` | Додати новий рядок у таблицю товарів |
| **Завантажити фото** | `⌘` + `I` | `Alt` + `I` | Викликати діалог вибору файлу (чека) |

---

## 🖼️ Перегляд зображень (Zoom)

Коли відкрито перегляд чека або фото.

| Дія | macOS | Windows / Linux |
| :--- | :--- | :--- |
| **Збільшити (Zoom In)** | `⌘` + `+` | `Alt` + `+` |
| **Зменшити (Zoom Out)** | `⌘` + `-` | `Alt` + `-` |
| **Скинути (Reset)** | `⌘` + `0` | `Alt` + `0` |

---

> **Примітка:** Ми використовуємо клавішу `Alt` для Windows/Linux, щоб уникнути конфліктів зі стандартними браузерними скороченнями (наприклад, `Ctrl + A` — виділити все).
